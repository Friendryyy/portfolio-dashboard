## 🏗️ แผนผังการทำงานของระบบเครื่องจักรอัตโนมัติ (Automation Architecture)

## 📅 ตารางบันทึกสถานะงานตั้งเวลาเชิงรุก (Active Schedules)

ระบบงานอัตโนมัติได้รับการย้ายมารันบน **Investment OS - Scheduler Daemon** ในระดับ OS ของคอมพิวเตอร์คุณโดยตรง (ไม่ต้องพึ่งพาหน้าต่างแชต AntiGravity ทำให้งานรันได้จริงตราบใดที่คอมพิวเตอร์เปิดอยู่และเชื่อมต่ออินเทอร์เน็ต) โดยระบบจะรันงานเบื้องหลังเงียบ ๆ ตลอดเวลาผ่านไฟล์คุมเวลาและไฟล์สคริปต์ PowerShell ทั้ง 4 ตัวนี้:

### 1. 📡 ระบบวิเคราะห์ภาพรวมพอร์ตโฟลิโอรายวัน (Daily Portfolio CMO)
*   **ไฟล์สคริปต์:** `scheduler/task_daily_portfolio.ps1`
*   **วัน/เวลาทำงาน:** ทุกวัน เวลา **12:00 น.** (เที่ยงตรงตามเวลาไทย)
*   **วัตถุประสงค์หลัก:** ดึงข้อมูลสัดส่วนพอร์ตโฟลิโอสด ๆ จาก Google Sheets API จากนั้นสั่งรัน Custom Sub-Agents Swarm (Macro, Fundamental, Technical, Risk) วิเคราะห์คู่ขนานกัน และผ่านด่านตรวจ QA Audit ของ Agent 14 (คะแนน ≥ 95) บันทึกรายงานวิเคราะห์ลง Obsidian `Database/` และ `output/` และ NotebookLM (ทั้ง Stock Notebook และ Master Hub) อัตโนมัติ

### 💀 2. เครื่องจับกระแสข่าวเดือดวิกฤตล่าโซน DCA ช้อนซื้อ (Sentiment Crisis Hunter)
*   **ไฟล์สคริปต์:** `scheduler/task_sentiment_hunter.ps1`
*   **วัน/เวลาทำงาน:** วันจันทร์ - วันศุกร์ เวลา **12:00 น.** (เที่ยงตรง)
*   **วัตถุประสงค์หลัก:** ยิง API ตรวจสอบ Fear & Greed Index ล่าสุด หากดัชนีเข้าสู่ระดับ Extreme Fear (≤ 25) หรือพบสัญญาณราคาหุ้นในพอร์ตร่วงหล่นต่ำกว่าแนวรับสำคัญ ระบบจะสปอว์น Sub-Agents ทำการประเมิน Intrinsic Value และ DCA Entry Zones ทันที เพื่อออกรายงาน alert ชี้ช่องช้อนซื้อในโฟลเดอร์ `output/`

### 🛍️ 3. ตัวช่วยจัดพอร์ตช้อปปิ้งส่งท้ายสัปดาห์ (Weekly DCA Shopping Assistant)
*   **ไฟล์สคริปต์:** `scheduler/task_weekly_dca.ps1`
*   **วัน/เวลาทำงาน:** ทุกวันศุกร์ เวลา **22:00 น.** (ก่อนปิดตลาดหุ้นสัปดาห์)
*   **วัตถุประสงค์หลัก:** ตรวจสอบความแข็งแกร่งทางเทคนิค Snapshot RSI และ SMA สแกนหา Bullish Divergence เพื่อคัดกรองหุ้น DCA ที่คุ้มค่าที่สุด จากนั้นสั่ง Swarm ออกรายงานแนะนำการซื้อรายสัปดาห์ จัดสรรงบประมาณ (เช่น $500/สัปดาห์) และแนวโน้มพอร์ตระยะยาว

## 🛠️ วิธีการบำรุงรักษาและการรันแบบกึ่งอัตโนมัติ (Daemon Operations & Controls)

ระบบรันอยู่เบื้องหลังเงียบ ๆ โดยใช้พลังงานและ Token ต่ำที่สุด คุณสามารถควบคุมระบบผ่านเครื่องมือปกติในโฟลเดอร์ `scheduler/` ได้ดังนี้:

### 1. วิธีเริ่มใช้งานครั้งแรก / การรันอัตโนมัติเมื่อเปิดเครื่อง
*   **การเชื่อม Windows Startup:** สคริปต์ติดตั้งจะทำการสร้าง Shortcut ของ `scheduler/start_daemon.bat` เข้าไปในโฟลเดอร์ Startup ของ Windows (`shell:startup`) เรียบร้อยแล้ว
*   **ผลลัพธ์:** ทุกครั้งที่คุณเปิดคอมและเข้าสู่ระบบ Windows โปรแกรม `start_daemon.bat` จะตื่นขึ้นมารัน Python Daemon เบื้องหลังเงียบ ๆ ในรูปแบบ Minimized (ย่อหน้าต่างลง Taskbar) คอยตรวจดูเวลาและทำงานอัตโนมัติ โดยที่คุณ**ไม่ต้องคลิกหรือพิมพ์คำสั่งใด ๆ เลย**

### 2. วิธีรันงานด้วยตนเองทันที (Manual Trigger)
หากคุณต้องการสั่งรันสคริปต์เพื่อดูรายงานทันที ณ ตอนนั้น โดยไม่รอตามรอบเวลาปกติ:
1. เปิด PowerShell (ไม่ต้องใช้สิทธิ์ Admin)
2. สั่งรัน PowerShell สคริปต์ตัวที่ต้องการได้เลย เช่น:
   ```powershell
   powershell -ExecutionPolicy Bypass -File scheduler/task_daily_portfolio.ps1
   ```
3. รอสักครู่ รายงานวิเคราะห์ตัวล่าสุดจะถูกบันทึกเข้าโฟลเดอร์ `output/` และพร้อมอัปเดตขึ้นบนแดชบอร์ดให้คุณทันที!

### 3. วิธีปิด/เปิด Daemon ด้วยตนเอง
*   **ปิดระบบ:** มองหาหน้าต่าง CMD บน Taskbar ที่ชื่อ `Investment OS — Scheduler Daemon` แล้วกด `Ctrl + C` หรือกดปุ่มกากบาท `[X]` เพื่อปิด
*   **เปิดระบบใหม่:** ดับเบิลคลิกไฟล์ `scheduler/start_daemon.bat` ในโฟลเดอร์ระบบได้ทันที
